================================================================================
                            DATABASE STRUCTURE
================================================================================

5 Supabase Tables Created:

1. employees
   - Phone, Employee ID, Name, Email
   - Department, Designation, Salary
   - DOB, Work Anniversary (NEW)
   - Profile picture URL
   - Manager name

2. attendance_logs
   - Employee ID, Date
   - Clock In time + GPS (lat/lng/accuracy)
   - Clock Out time + GPS
   - Hours worked, Status

3. leave_requests
   - Employee ID, Leave Type
   - From/To dates, Reason
   - Status (Pending/Approved/Rejected)
   - Approved by, Approved at

4. geofence_violations
   - Employee ID, Date, Time
   - GPS coordinates
   - Distance from office
   - Action taken

5. announcements
   - Title, Message
   - Created by, Created at
   - Expires at, Is active
